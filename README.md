# python_demo

- **python version 3.6.5**

- **重要：在一个项目下 所有的 modules 是一起编译的，也就是说不同的 modules 文件夹类名等不能重复！**

### 介绍

  1. code: python 语言入门
  2. game: python 项目练习
  3. matplotlib: python 统计分析库入门
  4. requests_demo: requests 网络请求库入门
  5. beautiful_soup_demo: bs4 网页解析库入门
  6. socket_demo: socket 库开发
